## Instalación

1. Clonar repositorio:
git clone https://github.com/TU_USUARIO/laravel-clase1.git

2. Entrar al proyecto:
cd laravel-clase1

3. Instalar dependencias:
composer install

4. Configurar entorno:
cp .env.example .env

5. Generar clave:
php artisan key:generate

6. Ejecutar migraciones:
php artisan migrate

7. Iniciar servidor:
php artisan serve



## Respuesta Preguntas

## 1: •	¿Qué diferencias y similitudes encuentras entre Django (Python) y un framework PHP? R:// Django y los frameworks PHP como Laravel son similares porque ambos se usan para desarrollo web, trabajan con bases de datos y siguen el patrón MVC. La diferencia principal es que Django usa Python y Laravel usa PHP; además, Django es más estructurado y Laravel más flexible.

## 2: •	¿Qué es Composer y para qué crees que sirve en un proyecto PHP? R:// Composer es una herramienta que sirve para instalar y gestionar librerías en proyectos PHP, facilitando el desarrollo y manteniendo todo organizado.

## 3: •	Si tuvieras que describir MVC a alguien que nunca ha programado, ¿cómo lo harías? R:// MVC es una forma de organizar un programa en tres partes: el modelo maneja los datos, la vista muestra la información al usuario y el controlador conecta todo y gestiona las acciones.
