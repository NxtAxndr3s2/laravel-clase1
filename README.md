## Instalación

1. Clonar repositorio:
git clone https://github.com/TU_USUARIO/laravel-clase1.git

2. Entrar al proyecto:
cd laravel-clase1

3. Instalar dependencias:
composer install

4. Configurar entorno:
cp .env.example .env

5. Generar clave:
php artisan key:generate

6. Ejecutar migraciones:
php artisan migrate

7. Iniciar servidor:
php artisan serve
